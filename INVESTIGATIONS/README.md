# SOC Home Lab

## Overview
This project is a home-based SOC (Security Operations Center) lab built to practice real-world cybersecurity skills such as log analysis, threat detection, and incident response.

## Tools Used
- Windows Virtual Machine
- Linux Virtual Machine
- Sysmon
- Splunk

## What I Practice Here
- Analyzing Windows Event Logs
- Investigating suspicious activity
- Detecting brute force attempts
- Searching logs using SIEM tools (Splunk)
- Writing incident reports

## Goal
To build hands-on experience that prepares me for a SOC Analyst role.
