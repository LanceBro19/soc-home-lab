# SOC Home Lab

This project is a home cybersecurity lab built to simulate real SOC (Security Operations Center) environments.

## Tools Used
- Windows Virtual Machine
- Linux Virtual Machine
- Sysmon
- Splunk

## Purpose
To practice log analysis, detect suspicious activity, and simulate real-world security investigations.

## Skills Practiced
- Windows Event Log analysis
- SIEM log searching (Splunk)
- Incident investigation
- Basic threat detection
